# TJA オフセット一括調整ツール (TJA Offset Shifter)

段位道場などの複数曲パック（`.tja` ファイル）の `OFFSET:` および `#DELAY` を、指定した値で一括調整できるブラウザ専用ツールです。

👉 **[ツールを使用する (Live Demo)](https://[ユーザー名].github.io/[リポジトリ名]/)**

## ✨ 特徴 (Features)

* **秒 / ミリ秒 (ms) の両対応:** 直感的にわかりやすい単位を選んでオフセットを調整できます。（例: 10ms = 0.01秒）
* **ドラッグ＆ドロップで簡単操作:** TJAファイルを放り込んで数値を指定するだけ。
* **上書き保存ライクな挙動:** ダウンロードされるファイル名は元のTJAファイルと同じになるため、そのままフォルダに上書きしてすぐにテストプレイ可能です。
* **OFFSET / #DELAY の個別設定:** どちらか片方だけをずらすことも、両方同時にずらすことも可能です。
* **ブラウザ完結:** サーバーへのファイルアップロードは行われません。すべて手元のブラウザ上で安全に処理されます。

## 使い方 (Usage)

1. [ツールの公開ページ](https://[ユーザー名].github.io/[リポジトリ名]/) にアクセスします。
2. 画面上の点線枠内に、調整したい `.tja` ファイルをドラッグ＆ドロップします。
3. **「ずらす値」** を入力します。（単位は 秒/ミリ秒 から選択可能）
   * **＋ / － の使い分け**
     * プレイしていて**遅延が気になる人** ➡️ プラス (+) に
     * 「この曲、**遅くないか？**」と思ったら ➡️ マイナス (-) に
4. 変更したい項目（`OFFSET:` / `#DELAY`）にチェックを入れます。
5. **「一括調整して保存」** ボタンをクリックします。
6. 同じファイル名でダウンロードされるので、元のファイルと置き換えてください。

## ⚠️ 注意事項 (Notes)

* 同じファイル名で出力されるため、念のため**元のTJAファイルはバックアップ**を取ってから作業することをおすすめします。
* シミュレータの仕様やバージョンによっては、UTF-8への変換により動作に影響が出る場合があります。
