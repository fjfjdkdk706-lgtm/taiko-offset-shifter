# TJA オフセット一括調整ツール (TJA Offset Shifter)

段位道場などの複数曲パック（`.tja` ファイル）の `OFFSET:` および `#DELAY` を、指定した秒数だけ一括で足し引き（ずらす）できるブラウザ専用ツールです。

「全体の音源のタイミングを変えたから、3曲まるごとオフセットをずらしたい」「曲間のDELAYの秒数もまとめて調整したい」といった場面で役立ちます。

👉 **[ツールを使用する (Live Demo)](https://fjfjdkdk706-lgtm.github.io/taiko-offset-shifter/)**

## ✨ 特徴 (Features)

* **ドラッグ＆ドロップで簡単操作:** TJAファイルを放り込んで秒数を指定するだけですぐに変換できます。
* **OFFSET / #DELAY の個別設定:** どちらか片方だけをずらすことも、両方同時にずらすことも可能です。
* **小数点誤差の自動補正:** プログラム特有の計算誤差（例: `0.1 + 0.2 = 0.300000004`）を防ぎ、綺麗な数値で出力します。
* **ブラウザ完結:** サーバーへのファイルアップロードは行われません。すべて手元のブラウザ上で安全に処理されます。
* **文字化け対策済み:** 出力ファイルはBOM付きUTF-8で保存されるため、日本語タイトルなどの文字化けを防ぎます。

## 使い方 (Usage)

1. [ツールの公開ページ](https://[ユーザー名].github.io/[リポジトリ名]/) にアクセスします。
2. 画面上の点線枠内に、調整したい `.tja` ファイルをドラッグ＆ドロップします。
3. **「ずらす秒数」** を入力します。
   * 全体的に譜面を**後ろに（遅く）** ずらしたい場合 ➡️ プラスの値（例: `0.05`）
   * 全体的に譜面を**前に（早く）** ずらしたい場合 ➡️ マイナスの値（例: `-0.02`）
4. 変更したい項目（`OFFSET:` / `#DELAY`）にチェックを入れます。
5. **「一括調整して保存」** ボタンをクリックします。
6. 変換結果のログが表示され、自動的に `元のファイル名_shifted.tja` という名前でファイルがダウンロードされます。

## ⚠️ 注意事項 (Notes)

* 本ツールは上書き保存ではなく別名保存を行いますが、念のため元のTJAファイルはバックアップを取ってから作業することをおすすめします。
* シミュレータの仕様やバージョンによっては、UTF-8への変換により動作に影響が出る場合があります。
